Extract and drop into eternal/mods (sorry for the zip file the site forced me to use some sort of compressed file type)

Currently supported on NIGHTLY only! It might work on stable but it is untested.

Pages - adding next/prev page for character select, additionally push right on last char for next page, left on first char for last page
Searching - search for chars by name
Tile scaling - Dynamically change the size of the char tiles in character select between 3 different sizes.

Styling depends on your motif see screenshots for example of vanilla vs custom
All features are enabled by default - disable/enable functions in the config section at the top
Additional configuration options for features in the config section as well

Keys should say on screen what they are for keyboard

Search - F3/Controller Start button
Tile size - F5/F6 Controller L1+L2
Paging - Next page L1 Prev Page L2 - Pressing right on last char in grid moves to next page, pressing left on first char on grid moves to prev page.
